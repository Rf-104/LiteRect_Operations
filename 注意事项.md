# 注意事项

1. 目前支持绝大部分快捷方式（***快捷方式***，***.exe***，***internet快捷方式***等），其他文件如***.doc***，***.txt***等常见文件也可以自动生成快捷方式存入绑定文件夹，如果有拖入不显示的图标请联系开发者

   **注意：软件图标拖入原理是自动在绑定文件夹建立快捷方式进行绑定，因此千万不要直接在绑定文件夹中放重要文件，和快捷方式以外的文件，不仅不会在窗口中显示，删除时还会一并删除**

2. 双击任务栏托盘会使所有窗口隐藏，但是这种隐藏方式和设置中***隐藏所有窗口***选项是两种不同的隐藏策略，请注意分辨，不建议同时开启，容易找不到窗口

3. 正常情况下点击设置中的***关于***按键会在窗口右下方跳出一个信息窗口；如果没有跳出，请检查电脑设置中的通知策略。*关于*按键不影响正常使用，因此即使没有跳出窗口也没有关系

4. 单窗口情况下唯一窗口无法删除

5. 有任何问题可以通过点击设置中的***刷新视图***进行刷新；如果没有用，在**已知问题解决方案**中查找相关解决办法；如果已知问题解决方案没有收录问题，请联系开发者

6. 从2.0.4.0版本开始，所有配置文件（config.ini，默认绑定文件夹）均转移至 ~~C:\Users\用户名\AppData\Local\LiteRect~~ 下

   4/10 注：对于**微软商店**版本，受**微软商店**的软件沙盒机制以及文件系统访问受限的问题影响，以上路径失效，同时存在**已知问题**文档中第3个问题，目前当微软商店设置下载地址盘符为C时，默认配置文件路径为：C:\Users\用户名\AppData\Local\Packages\Rf104.LiteRect_5cse80b1c4wsa\LocalCache\Local\LiteRect
   
   **该问题不影响其余功能使用**
